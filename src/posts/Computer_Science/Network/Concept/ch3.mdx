4. Describe why an application developer might prefer to run an application over UDP rather than TCP.<br/>
	응용 프로그램 개발자가 TCP 대신 UDP를 통해 응용 프로그램을 실행하는 것을 선호하는 이유를 설명하십시오.<br/>
	<Answer>
		응용 프로그램 개발자가 UDP 대신 TCP를 통해 응용 프로그램을 실행하는 이유는 다음과 같습니다:
		- **데이터 전송 속도** - UDP는 데이터 전송 속도가 빠르기 때문에 실시간 데이터 전송이 필요한 응용 프로그램에서 UDP를 사용할 수 있습니다.
		- **데이터 전송 보장** - UDP는 데이터 전송을 보장하지 않기 때문에 데이터 전송 보장이 필요하지 않은 응용 프로그램에서 UDP를 사용할 수 있습니다.
		- **연결 설정** - UDP는 연결 설정 과정이 없기 때문에 연결 설정 과정이 필요 없는 응용 프로그램에서 UDP를 사용할 수 있습니다.
	</Answer>
