# 확률 변수에 대한 조건화
  - 사건 A에 대해 조건화된 확률 변수 X의 조건부 확률 질량 함수
    - 사건 A에 대해 조건화된 확률 변수 X의 조건부 확률 질량 함수는 $P(A) > 0$일 때 다음과 같이 정의된다.
      - $p_{X|A}(x) = P(X = x | A) = \frac{P(\{X = x\} \cap A)}{P(A)}$

### 추가 설명

- 조건부 확률 질량 함수는 특정 사건 A가 주어졌을 때 확률 변수 X의 값을 찾는 데 사용된다.
- 조건부 확률은 사건 A가 발생한 상황에서 확률 변수 X가 특정 값을 가질 확률을 나타낸다.
- 공식에서 $P(\{X = x\} \cap A)$는 X가 x 값을 가지면서 동시에 사건 A가 발생하는 확률을 의미한다.
- $P(A)$는 사건 A의 발생 확률을 나타낸다.
- 따라서, 조건부 확률 질량 함수는 이 두 확률의 비율로 계산된다.
- 이는 확률 변수의 행동이 다른 사건의 발생에 의해 어떻게 영향을 받는지 이해하는 데 중요하다.

# 다른 확률 변수에 대한 조건화
  - 확률 변수 X와 Y가 동일한 실험과 관련되어 있을 때, Y에 대해 조건화된 X의 조건부 확률 질량 함수를 정의한다.
    - $p_{X|Y}(x | y) = P(X = x | Y = y)$
  - 이는 사건 A의 형태 $\{Y = y\}$로 정의의 특수화와 같다.
  - 조건부 확률의 정의를 사용하면 다음과 같다.
    - $p_{X|Y}(x | y) = \frac{P(X = x, Y = y)}{P(Y = y)} = \frac{p_{X,Y}(x, y)}{p_Y(y)}$

### 추가 설명

- 이 슬라이드는 하나의 확률 변수가 다른 변수에 의존하는 조건 하에서의 분포를 계산하는 방법을 설명한다.
- 조건부 확률 질량 함수 $p_{X|Y}(x | y)$는 확률 변수 Y가 특정 값 y를 갖는 조건 하에서 X가 x 값을 갖는 확률을 나타낸다.
- 이 계산은 결합 확률 분포 $p_{X,Y}(x, y)$에서 Y의 주변 확률 $p_Y(y)$로 나눔으로써 이루어진다.
- 이러한 분석은 통계적 연구에서 두 변수 간의 관계를 이해하고 예측하는 데 중요한 도구로 사용된다.
- 예를 들어, 학생들의 성적 분포를 연구할 때, 특정 학년의 학생들만을 대상으로 그들의 성적 분포를 조사하고자 할 때 조건부 확률을 사용할 수 있다.


# 예제
## 예제 1
### 문제
- X를 주사위를 굴려 나온 수라고 하고, A를 주사위의 눈금이 짝수인 사건이라고 하자.
- 앞서 언급한 공식을 적용하면 다음과 같은 결과를 얻는다.
	- $p_{X|A}(x) = P(X = x | \text{주사위 눈금이 짝수})$
	- $= \frac{P(X = x \text{ 그리고 } X \text{가 짝수})}{P(\text{주사위 눈금이 짝수})}$
	- $= \left\{\begin{array}{ll}1/3 & \text{if } x = 2, 4, 6, \\0 & \text{otherwise}.\end{array}\right.$

### 추가 설명

- 이 예에서 X의 가능한 값은 주사위의 눈금인 1, 2, 3, 4, 5, 6이다.
- 짝수가 나올 사건 A의 확률은 $\frac{3}{6} = \frac{1}{2}$이다, 왜냐하면 2, 4, 6 세 개의 짝수가 가능하기 때문이다.
- 조건부 확률 $P(X = x | \text{주사위 눈금이 짝수})$는 X가 2, 4, 6 중 하나일 때만 계산되며 각각의 확률은 $1/3$이다.
- X가 1, 3, 5와 같이 짝수가 아닌 값일 때는 조건부 확률이 0이 된다, 왜냐하면 이 값들은 사건 A (주사위 눈금이 짝수)와 호환되지 않기 때문이다.
- 이 예제는 조건부 확률이 어떻게 특정 사건에 기반하여 확률 변수의 분포를 변형시키는지 보여준다.
