# JS
### 자바스크립트는 무슨 언어인가?
자바스크립트는 본래 브라우저를 제어하는 언어였으나, Node.js라는 새로운 실행환경으로 *자체 어플리케이션*을 만들 수 있게 된 언어이다.
또한 *명령형*, *함수형*, *객체지향 프로그래밍*이 모두 가능한 멀티 패러다임 언어이다.

### 변수 선언, 초기화, 할당의 차이점은?
변수 선언- 변수를 *생성*하는 것. 해당 식별자를 등록하여 스코프가 참조할 대상을 만든다.
변수 초기화- 메모리에 변수 *저장을 위한 공간* 확보(기본값: undefined)
변수 할당- 변수에 *값을 대입*하는 것. 이미 생성된 변수에 값을 대입하는 것이다.(기호: =)

### 데이터 타입에 대해 설명하시오
JS의 타입은 원시값과 객체로 나뉜다.
원시값(불변값): number, string, boolean, null, undefined, symbol 등
객체(참조값): *키*와 *값* 사이의 매핑. 키는 문자열이나 심볼이고 값은 모든 타입이 될 수 있다.
함수: 호출이 가능하다는 것을 제외하면 객체와 동일하다.

### this 키워드에 대해 설명하시오
*자신이 속한 객체*를 가리키는 식별자를 참조하는 키워드
함수가 호출되는 방식에 따라 this가 가리키는 대상이 달라진다.
1. 일반 함수 호출: *전역 객체*
	```js
	function foo() {
		console.log(this);
	}
	foo(); // window
	```
2. 메소드 호출: *마침표 연산자 앞*에 기술한 객체
	```js
	const obj = {
		foo() {
			console.log(this);
		}
	};
	obj.foo(); // obj
	```
3. 생성자 함수 호출: 생성자 함수가 *미래에* 생성할 인스턴스
	```js
	function Foo() {
		console.log(this);
	}
	new Foo(); // Foo {}
	```
4. 화살표 함수: 상위 스코프의 this를 가리킨다.
	```js
	const foo = () => console.log(this);
	foo(); // window
	```

### call, apply, bind 메소드에 대해 설명하시오
call, apply, bind는 *함수*의 메소드로 this를 *명시적으로(강제로)* 바인딩할 때 사용한다.
**call**과 **apply**는 함수를 호출하는 함수이지만 첫 번째 인자로 this로 사용할 객체를 전달한다.
```js
function foo() {
	console.log(this);
}
const obj = {};
foo.call(obj); // obj
foo.apply(obj); // obj
```
**bind**는 함수를 호출하는 함수이지만 this로 사용할 객체를 전달하고 새로운 함수를 반환한다.
```js
function foo() {
	console.log(this);
}
const obj = {};
const bar = foo.bind(obj);
bar(); // obj
```

### 콜백 함수에 대해 설명하시오
콜백 함수는 *다른 함수의 인자로 전달되는 함수*이다.
비동기 처리에서 주로 사용되며, 이벤트 핸들러나 타이머 함수, Ajax 요청, Promise 객체 등에서 사용된다.
```js
function foo(callback) {
	callback();
}
foo(() => console.log('Hello, World!'));
```

### 콜백 지옥을 해결하기 위한 방법은?
콜백 지옥은 비동기 처리 로직을 위해 콜백 함수를 연속저으로 중첩해 사용할 때 발생하는 문제이다.
콜백 지옥을 해결하기 위한 방법은 **Promise**, **async/await** 등이 있다.
- *Promise*는 비동기 연산이 종료된 이후에 결과 값과 에러를 처리할 수 있는 객체이다.

	Promise의 .then() 메소드를 사용하여 함수 실행 순서를 정할 수 있다.
- *async/await*는 Promise를 더 쉽게 사용할 수 있게 해주는 ES8 문법이다.

	async 함수는 Promise를 반환하고, await 키워드는 Promise가 처리될 때까지 기다린다.

```js
// Promise
function foo() {
	return new Promise((resolve, reject) => {
		setTimeout(() => resolve('Hello, World!'), 1000);
	});
}
foo().then(console.log); // Hello, World!

// async/await
async function bar() {
	const result = await foo();
	console.log(result);
}
bar(); // Hello, World!
```

# React
### React란 무엇인가?
React는 UI를 만들기 위한 *JS 라이브러리*이다.
스스로 상태를 관리하는 캡슐화된 컴포넌트를 조합해 복잡한 UI를 만들 수 있다.
데이터가 벼경될 때마다 *적절한 컴포넌트만 효율적으로* 갱신하며, 빠른 렌더링을 위해 가상 DOM을 사용한다.
