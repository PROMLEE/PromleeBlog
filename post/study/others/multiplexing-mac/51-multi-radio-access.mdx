## 개요 (Introduction)
다중 접근 네트워크
- 각 노드는 수신기에 연결되어 있으며, 다른 노드들과 공유되는 매체를 통해 통신함
- 어느 노드에서의 전송도 다른 모든 노드들에 의해 수신됨

## 다중 접근 (Multiple Access)
다중 접근 문제
- 여러 노드가 동시에 broadcast 채널에서 전송 시 충돌 발생
- 어떤 노드가 전송할 수 있는 지 결정하는 방법 필요
*다중 접근 프로토콜*
- 다중 접근 문제 해결을 위한 프로토콜

## 다중 접근 프로토콜 (Multiple Access Protocol)
1. 비경쟁 프로토콜 (Contention-Free Protocol)
	- 동적 할당 방식(Dynamic Allocation) - 예약 방식(Reservation)
	- 정적 할당 방식(Static Allocation)
2. 경쟁 기반 프로토콜 (Contention-Based Protocol)
3. 혼합 프로토콜 (Hybrid Protocol)
